NAME

    add_rdr - add a new rule to an RDR

SYNOPSIS

    add_rdr(+Rule)

DESCRIPTION

    Rule must be a term of the form:

      if condition then conclusion because case.

    Add_rdr must only be called after an RDR has been evaluated. iProlog keeps track of the last rule tested in an RDR. When add_rdr is called, the new rule is insert into the RDR after the last rule tested.

    • If the last rule succeeded, the new rule is added as an exception (except clause).

    • If the last rule failed, the new rule is added as an alternative (except clause).
