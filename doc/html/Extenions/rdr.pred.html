NAME

    rdr - perform maintenance on RDRs

SYNOPSIS

    rdr(+EvaluableTerm)

DESCRIPTION

    EvaluableTerm must be a term that can be evaluated as a function and is implemented as a ripple-down rule. Rdr invokes an interactive session:

    1. Rdr evaluates the expression and asks the user if the result is correct.
    2. If the answer is yes, rdr succeeds and the call terminates.
    3. If the answer is no, rdr asks the user for the correct answer.
    4. It then lists a sequence of tests to be used in building a new rule to correctly classify the new case.
    5. If the user answers yes to a test, then it is included as one of the conditions in the new rule.
    6. If the user answers no, the test is discarded.
    7. When the list of test has been exhausted, the new rule is installed in the RDR and the call the rdr succeeds.
