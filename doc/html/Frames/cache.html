NAME

    cache - a flag to indicate if an if_needed value should be stored.

SYNOPSIS

    SlotName:

      cache yes

DESCRIPTION

    Normally, when an if_needed dæmon is triggered, its value is returned and when the same dæmon is triggered again, the value is recomputed. If you wish the value to be remembered, add a cache flag to the slot. The next time and if_needed dæmon is triggered, its result will stored in the corresponding slot of the instance frame that invoked the if_needed dæmon.

EXAMPLE

    Some of the slots in the cylinder and patient examples have cache dæmons.
