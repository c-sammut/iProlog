NAME

    default - sets a default value for a slot

SYNOPSIS

    SlotName:

      default term

DESCRIPTION

    When a request for a slot value fails to find the value in the instance frame, the inheritance hierarchy is searched. The default dæmon sets a default value for a slot.

    Note: default has preceence over if_needed . That is, if a slot has both dæmons, the default value is selected. Also note that, unlike, if_needed, the term stored with the default facet is not evaluated.

EXAMPLE

    Some of the slots in the patient example have default dæmons.
