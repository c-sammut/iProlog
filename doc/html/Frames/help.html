NAME

    help - dæmon predicate invoked when range check fails

SYNOPSIS

    SlotName:

      help predicate

DESCRIPTION

    When the range dæmon, it checks to see if a help dæmon exists for the same slot. If it does, it is called. This is usually used to print an error message.

EXAMPLE

    Most of the slots in the cylinder and patient examples have range and help dæmons.
