NAME

    if_needed - dæmon function invoked when a computed value is needed

SYNOPSIS

    SlotName:

      if_needed predicate

DESCRIPTION

    When a request for a slot value fails to find the value in the instance frame, the inheritance hierarchy is searched. The if_needed dæmon computes a value.

    Note: the if_needed dæmon is a function. That is, it must be an expression that returns a value.

EXAMPLE

    Most of the slots in the cylinder and patient examples have if_needed dæmons.
