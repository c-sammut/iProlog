NAME

    multivalued - allow a slot to contain a list of vlaues

SYNOPSIS

    SlotName:

      multivalued yes

DESCRIPTION

    This must only appear as a facet of a slot in a generic frame. It allows more than one value to be added to a slot value.

EXAMPLE

    box ako object with
       contents:
               multivalued     yes!

: b isa box?
** yes
: add marble to contents of b?
** yes
: contents of b?
[marble]
: add string to contents of b?
** yes
: contents of b?
[string, marble]
: listing?

value(box, isa, [object]).
value(b, isa, [box]).
value(b, contents, [string, marble]).

multivalued(box, contents, yes).
