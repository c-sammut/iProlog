NAME

    slot-name - get the slot value of a frame

SYNOPSIS

    slot-name(object)

DESCRIPTION

    slot(frame) retrieves the value of slot from frame. If the value is not present in the instance frame then the inherited value is obtained. If there is no inherited value, the call fails.

    If slot-name appears without an argument, it is assumed that the slot value is to be found in the current object.

EXAMPLE

cylinder ako [] with
[
        height:
        [
                range           number(new value) and new value > 0,
                help            print("Height must be a positive number"),
                if_needed       ask_user,
                if_removed      remove volume,
                cache           yes
        ],
        radius:
        [
                range           number(new value) and new value > 0,
                help            print("Radius must be a positive number"),
                if_needed       ask_user,
                if_removed      remove cross_section,
                cache           yes
        ],
         cross_section:
        [
                if_needed       pi * radius ** 2,
                if_removed      remove volume,
                cache           yes
        ],
         volume:
        [
                if_needed       cross_section * height,
                cache           yes
        ]
 ]!
c isa [cylinder] with
[
       height: 2,
       radius: 4
]!

: height(c)?
2
: volume(c)?
100.531
colour(c)?
** no

When the value of volume is requested, iProlog first searches the instance frame c for a slot named volume. When this is not found it checks c's isa slot and looks for volume in the generic cylinder frame. When found, the if_needed dæmon is triggered.
