NAME

    range - dæmon predicate applied to a new value when added

SYNOPSIS

    SlotName:

      range predicate

DESCRIPTION

    The range dæmon is invoked when a new value is added to a slot, either by the add or replace predicates. The dæmon is implemented by a predicate which is applied to new value.

    • If the predicate succeeds, the new value is added.
    • If the predicate fails, and a help dæmon is also present, then the help dæmon is invoked. Nothing is added when the range dæmon fails.

EXAMPLE

    Most of the slots in the cylinder and patient examples have range and help dæmons.
