NAME

    remove - remove a slot in a frame

SYNOPSIS

    remove slot(frame)

DESCRIPTION

    A predicate that removes a slot from a frame.

    Triggers the if_removed dæmon.

EXAMPLE

    Given the generic definition of cylinder, some instances can be defined as:

    c isa [cylinder] with [height: 2, radius: 4]!
remove radius(c)?
