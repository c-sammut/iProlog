NAME

    replace - replace the value of a slot in a frame

SYNOPSIS

    slot(frame) is_replaced_by value

DESCRIPTION

    A predicate that replaces the current value of a slot with a new value. If the slot does not exists, replace is the same as add.

    If a slot has been declared multivalued, a replace replaces all values by the new value. In this case, the new value should be a list.

    Triggers the if_replaced dæmon.

EXAMPLE

    Given the generic definition of cylinder, some instances can be defined as:

    c isa [cylinder] with [height: 2, radius: 4, colour: red]!
colour(c) is_replaced_by blue?
