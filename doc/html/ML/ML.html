Machine Learning Tools

The most important distinguishing feature of iProlog is that it incorporates a number of tools for machine learning.

Propositional Learning

  • Aq
  • Decision Tree Induction
  • Induct/RDR

  • Duce

Statistical Methods

  • Naive Bayes
  • Feed-forward neural net with back-propagation
  • Linear Discriminant

  • Regression Trees

First-order Learning

  • Least General Generalisation

  • Refinement Rules
