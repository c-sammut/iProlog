NAME

    =.. - univ

SYNOPSIS

    +Term =.. ?List

    ?Term =.. +List

DESCRIPTION

    List is a list whose head is the atom corresponding to the principal functor of Term and whose tail is the argument list of that functor in Term.

    If Term is instantiated to a variable, then List must be instantiated to a list of determinate length whose head is atomic (ie. an atom or integer).

EXAMPLE

      product(0, N, N - 1) =.. [product, 0, N, N - 1]
