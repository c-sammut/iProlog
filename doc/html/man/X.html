NAME

    call - call a predicate bound to a variable

SYNOPSIS

    X

DESCRIPTION

    If X is boundto a term which would be acceptable as the body of a clause, the goal X is executed exactly as if that term appeared in place of the X .

    Any cut (`!') occurring in X is interpreted as if it occurred in the body of the clause containing X .

ERRORS

    If X is not instantiated as described the clause fails.
