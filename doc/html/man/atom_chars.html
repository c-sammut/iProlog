NAME

    atom_chars - interconvert atoms and lists of characters

SYNOPSIS

    atom_chars(+Atom, ?List)
    atom_chars(?Atom, +List)

DESCRIPTION

    If Atom is instanitated, List is unified with the list of single character atoms that make up Atom's name.

    If List is instantiated to a list of single character atoms, they are combined to form a single atom which is unified with Atom.

EXAMPLE

    atom_chars(abc, [a, b, c])?

COMPATIBILITY

    iProlog implements an extended form of atom_chars in which the elements of the instantiated List, may be atoms of any kind, eg:

    : atom_chars(X, [fred, 123, xyz])?
    X = fred123xyz
