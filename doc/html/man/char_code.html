NAME

    char_code - the ascii (or whatever) code for a character

SYNOPSIS

    char_code(+Char, ?Code)
    char_code(?Char, +Code)

DESCRIPTION

    Code is the ascii (or whatever code is use by your OS) equivalent of the character Char, where Char is a one-character atom.

    If Char is a single character atom, Code is instantiated to the ascii code representing Char.

    If Char is a variable and Code is an integer, Char is instantiated to the appropriate single character atom.
