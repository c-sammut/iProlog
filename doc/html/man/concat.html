NAME

    concat - concatenate a list of atomic elements

SYNOPSIS

    concat(+List, ?Atom)

DESCRIPTION

    The members of list List are concatenated to form the atom Atom.

    The members of List must be atoms or integers. If Atom is instantiated then its value is compared with the result of the concatenation.

EXAMPLES

        concat([abc, def],  X)?
        X = abcdef

        concat([f, r, e, d, 12], X)?
        X = fred12

COMPATIBILITY

    Concat is not in the ISO Prolog Standard
