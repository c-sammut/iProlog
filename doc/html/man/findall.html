NAME

    findall - find all solutions

SYNOPSIS

    findall(+V, +P, -L)

DESCRIPTION

    Find all terms, V, such that P is true. Append V to the list, L. Findall uses backtracking to find all the solutions to P.

NOTE

    This is the same as ISO Prolog's bagof predicate except that the additional "^" notation for existentially quantified variables is not required. All unbound variables in P are assumed to be existentially quantified.
