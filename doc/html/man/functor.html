NAME

    functor

SYNOPSIS

    functor(+Term, -Functor, -Arity)

    functor(-Term, +Functor, +Arity)

DESCRIPTION

    The principal functor of term Term has name Functor and arity Arity, where Functor is either an atom or, provided Arityis 0, an integer.

    Either Term must be instantiated to a non-variable, or Functor and Arity must be instantiated to, respectively, either an atom and a non-negative integer or an integer and 0. If these conditions are not satisfied, an error message is given.

    In the case where Term is initially instantiated to a variable, the result of the call is to instantiate Term to the most general term having the principal functor indicated.
