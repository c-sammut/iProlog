NAME

    load - read a Prolog program

SYNOPSIS

    load +FileName
    load +ListOfFileNames

DESCRIPTION

    FileName must be the name of a file containing a Prolog program. Clauses in the file are read and interpreted just as they would be if typed directly to the interpreter.

    If the file had been loaded previously, the previous clauses would have been removed first, to be replaced by the new ones.

    If the argument is a list of file names, each file is loaded in order.

    A side-effect of loading a file is that Prolog asserts a clause of the form:

      file(FileName, ProcList)

    into the database. ProcList is a list of the names of procedures defined in FileName. This list is used to unload procedures before being reloaded.
