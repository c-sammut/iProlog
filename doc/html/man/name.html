NAME

    name - convert atoms to list of characters and vice-versa

SYNOPSIS

    name(+Atomic, -CharList)
    name(-Atomic, +CharList)

DESCRIPTION

    If Atomic is an atom, integer or real then CharList is a list of the characters of the name of Atomic .

    If Atomic is instantiated to a variable, CharList must be instantiated to a list of characters.

EXAMPLES

      name(product, [p, r, o, d, u, c t])
      **yes
      name(X, [f, r, e, d]).
      X = fred
