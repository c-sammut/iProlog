NAME

    number_chars - interconvert numbers and lists of characters

SYNOPSIS

    number_chars(+Number, ?List)
    number_chars(?Number, +List)

DESCRIPTION

    If Numberis instanitated, List is unified with the list of single character atoms that make up Number's digits.

    If List is instantiated to a list of single character atoms, they are combined to form a single number which is unified with Number.

EXAMPLE

    number_chars(123.4, ['1', '2', '3', '.', '4'])?
