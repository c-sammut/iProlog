NAME

    numbervars - make a non-ground term ground

SYNOPSIS

    numbervars(+Term, +FirstVar, ?LastVar)

DESCRIPTION

    Unifies each of the variables in term Term with a special term, so that 'write(Term)' prints each of those variables as "_I", where the Is are consecutive integers from FirstVar to LastVar - 1.

    FirstVar must be bound to an integer.
