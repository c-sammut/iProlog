NAME

    unload - procedures defined in a file

SYNOPSIS

    unload +FileName
    unload +ListOfFileNames

DESCRIPTION

    Retracts all procedures defined in FileName (according to the contents of file(FileName, ProcList) Also retracts file(FileName, ProcList) .

    If the argument is a list of file names, unload retracts the procedures defined in each file.
